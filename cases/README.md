# Case Studies

Real and illustrative teams run through Stratum, showing what the framework surfaces.

| Case | Type | Notes |
|------|------|-------|
| [The Meridian Pod](./meridian_pod_illustrative_case_2026-07-06.md) | **Illustrative** | Fictional 4-person pod (the `cli/simulate.js` team). Hand-authored scores that probe specific failure modes. Not a real client. |

Illustrative cases use designed personas so the framework's behaviour is legible and
reproducible (`node cli/generate-samples.js`). They make no claim of real-world deployment.
Validated case studies with outcome data would live here too — none exist yet, and the
[Evidence Ledger](../ledger/) says so plainly.
